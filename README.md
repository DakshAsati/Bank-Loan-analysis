# 💳 Bank Loan Analysis Dashboard

An end-to-end **Bank Loan Analysis** project built using **SQL Server** and **Power BI** to analyze loan applications, borrower behavior, loan performance, and business KPIs. The dashboard provides actionable insights to help financial institutions make data-driven lending decisions.

---

## 📌 Project Overview

This project focuses on analyzing bank loan data to evaluate lending performance, identify trends, monitor loan quality, and understand borrower characteristics.

The solution includes:
- Data Cleaning
- SQL Analysis
- Power BI Dashboard
- KPI Reporting
- Business Insights

---

## 🚀 Objectives

- Analyze loan application trends
- Monitor funded and received loan amounts
- Compare Good Loans vs Bad Loans
- Track monthly lending performance
- Evaluate borrower financial health
- Support business decisions using interactive dashboards

## 📊 Dashboard Features

### 1️⃣ Summary Dashboard

Key Performance Indicators (KPIs)

- Total Loan Applications
- Total Funded Amount
- Total Amount Received
- Average Interest Rate
- Average Debt-to-Income (DTI)
- Month-to-Date (MTD)
- Month-over-Month (MoM)

Good Loan Analysis

- Good Loan Percentage
- Good Loan Applications
- Good Loan Funded Amount
- Good Loan Amount Received

Bad Loan Analysis

- Bad Loan Percentage
- Bad Loan Applications
- Bad Loan Funded Amount
- Bad Loan Amount Received

---

### 2️⃣ Overview Dashboard

Interactive visualizations include:

- 📈 Monthly Loan Trends
- 🗺 Regional Analysis by State
- 🍩 Loan Term Distribution
- 📊 Employment Length Analysis
- 💰 Loan Purpose Breakdown
- 🏠 Home Ownership Analysis

---

### 3️⃣ Details Dashboard

Comprehensive loan-level information including:

- Loan ID
- Borrower Information
- Loan Status
- Purpose
- Interest Rate
- Funded Amount
- Amount Received
- Debt-to-Income Ratio
- Employment Length
- State

---

## 📈 Key Business Insights

- Loan applications vary across different months.
- Regional lending patterns highlight high-performing states.
- Employment history influences loan approvals and repayments.
- Good loans contribute significantly to overall revenue.
- Higher DTI borrowers generally represent higher lending risk.

---

## 💡 Business Recommendations

- Improve credit risk assessment.
- Monitor bad loan percentage regularly.
- Focus on high-performing regions.
- Optimize loan approvals using borrower DTI.
- Track monthly lending KPIs for better forecasting.
